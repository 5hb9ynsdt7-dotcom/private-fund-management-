## 回滚指南

### 快速回滚到v1.0.0
1. 停止当前服务
2. 恢复项目文件：`cp -r backups/v1.0/* ./`
3. 恢复数据库：`cp backups/v1.0/*.db backend/`
4. 重启服务

### 数据回滚
```bash
# 备份当前数据
cp backend/privatefund_dev.db backend/privatefund_dev_backup.db

# 恢复到指定版本
cp backups/v1.0/privatefund_dev_v1.0_*.db backend/privatefund_dev.db
```
